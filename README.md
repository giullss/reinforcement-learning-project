# reinforcement-learning-project
questa reposity è basata sull'ambiente di lavoro dell'università di Berkeley (link della repo: https://github.com/jeff-hykin/berkeley_pacman.git) 
Contiene un progetto di RL dove sono stati realizzati due modelli 
  1) Q-learning tabellare
  2) Double DQN
i modelli verranno valutati in base al tipo di fantasma (random o directional), se è presente la possibilità di usare i power up.
## contenuto
il file di Colab è contenuto nella cartella notebook, per runnare il codice è necessario:
  1) aprire il notebook "ML_finale.ipynb" (di seguito anche il link: https://colab.research.google.com/drive/12-80epEHLBLDoOJhwwGRjS_32l8xNrKX) 
  2) runnare le celle in ordine sequenziale
