# reinforcement-learning-project
Questa repository è basata sull'ambiente di lavoro dell'Università di Berkeley  
(link della repo: https://github.com/jeff-hykin/berkeley_pacman.git).

Contiene un progetto di Reinforcement Learning sviluppato in ambito universitario, in cui sono stati implementati e confrontati due modelli:

  1. Q-learning tabellare  
  2. Double DQN  
i modelli verranno valutati in base al tipo di fantasma (random o directional), se è presente la possibilità di usare i power up.
## contenuto
il file di Colab è contenuto nella cartella notebook, per runnare il codice è necessario:
  1. Aprire il notebook `ML_finale.ipynb`  
   (link Colab: https://colab.research.google.com/drive/12-80epEHLBLDoOJhwwGRjs_32I8xNrKX)
  2. Eseguire le celle in ordine sequenziale
